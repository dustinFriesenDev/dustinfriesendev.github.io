Trailblazers Project

Trailblazers was built by a team of developers and myself that enables users to discover Missouri state parks, providing information on locations, weather, and local flora and fauna. This app allows users to record their trips, log the wildlife they saw, and share their experiences on a dedicated forum.
Tech Stack: Spring Boot, React, MySQL, Discourse, MapBox, and Auth0

home page

The home page loads a Missouri state map that call the MapBox API to render all the state parks and historical sites.

Logged in page

After logging in the nav bar is updated to include the users Trips taken and the user's profile information.

Plants page

The plants page can be visited before or after logging in. It display a list of 52 plants that are found in Missouri state wide and not specific to any locaiton. There is no current database of plants online that shows the local plant life specific to a park or site.

Clicking on the name of the plant on the left will show the details on the right.

trip page top

The trip page is onlly available after logging in. This page allows the user to provide a name, location, date, and any notes for the trip in the input boxes. The middle container shows plants or animals and can be toggled by the button above the middle column. Here the user can add a plant to an array that will be connected to the trip or delete the plant's id from the array with the add or delete buttons. An image is shown on the right when the text is clicked.

Trip page bottom

At the botom of the trip page, shows the trips the user has recorded as cards with an update button that repopulates the left and middle containers or the user can delete the card and is prompted to confirm the user wants to delete the trip.
